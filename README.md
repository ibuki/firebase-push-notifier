# Firebase push notifier

FirebaseでPUSH通知を行うシステム

# 動作確認方法

```
npm install -g firebase-tools
```

```
firebase serve
```

- functionsがコンパイルされないことがあるので注意。functionsディレクトリでnpm run buildする必要があるかも。

# デプロイ方法

```
firebase deploy
```
